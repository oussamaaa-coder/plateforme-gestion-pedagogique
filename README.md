## À propos

Cette plateforme web a été conçue et développée pour répondre aux besoins pédagogiques et administratifs de **l'ISTA NTIC SYBA**.

Elle centralise les informations académiques et administratives dans une seule application afin de faciliter la gestion des étudiants, des formateurs, des filières, des groupes et des ressources pédagogiques.

La plateforme offre une interface moderne, responsive et sécurisée destinée aux différents acteurs de l'établissement.

## Installation

### Cloner le projet

```bash
git clone https://github.com/oussamaaa-coder/plateforme-gestion-pedagogique.git
```

### Installer le Backend

```bash
cd backend

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```

### Installer le Frontend

```bash
cd frontend

npm install

npm run dev
```
